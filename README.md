# YouTube Content Factory

YouTube Content Factory automates the creation of YouTube video assets. By entering a topic, title, and notes for both long-form and short-form content, the system generates thumbnail prompts, formatted images, detailed descriptions, hashtags, SEO keywords, and social media posts. It streamlines daily content creation for both long videos and shorts.

## Features

* Input topic, title, and notes for both long videos (16:9) and shorts (9:16).
* Generates thumbnail prompts for AI image generation.
* Produces YouTube descriptions (both Tamil and English).
* Provides hashtags and SEO keywords.
* Outputs social media post drafts for community, Facebook, and Instagram.

## How to Use

1. Enter long video and shorts topics, titles, and notes.
2. Run the generation process (via the API).
3. Receive thumbnail prompts, images, descriptions, hashtags, and tags.
